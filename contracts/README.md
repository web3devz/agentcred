# contracts

Solidity contracts for AgentCred on Base Sepolia.

## Included
- `EscrowAgreement.sol`: milestone escrow lifecycle with submission/approval/release/dispute/refund
- `ReputationRegistry.sol`: updater-gated reputation writes + completion counters
- `AgentReceiptRegistry.sol`: receipt hash event registry
